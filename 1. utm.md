# UTM 설치 요약 (macOS)

> 목표: macOS에서 UTM으로 K3s 멀티노드 VM(cp1/w1/w2)을 준비합니다.

---

## 1) 다운로드/설치

- 다운로드: <https://mac.getutm.app/>
- Apple Silicon이면 **ARM64 Ubuntu ISO**를 사용합니다.

---

## 2) VM 생성 요약

- UTM에서 Ubuntu VM 3대(cp1/w1/w2) 생성
- CPU/메모리는 동일하게 맞추는 것을 권장

---

## 3) 네트워크 설정

각 VM → **Settings → Network**

- **Network Mode**: Shared Network
- (권장) Adapter 1 하나만 사용

---

## 4) 다음 단계 (미리보기)

VM 내부에서 `192.168.64.10/11/12` 정적 IP를 설정하면 K3s 멀티노드 구성이 가능합니다.  
서브넷이 다르면 그 값으로 치환하세요.
